
# Unit Conversion

A simple Unit Converter using the JavaFX library. 

### Conversion Categories
| **Length** | **Temperature** | **Mass** |
| ---------- | --------------- | -------- |
| Inch       | Fahrenheit      | Milligram|
| Foot       | Celsius         | Gram     |
| Yard       | Kelvin          | Kilogram |
| Mile       |                 | Metric Ton|
